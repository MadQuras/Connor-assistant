Connor Assistant — Помощь

Версия приложения определяется содержимым `package.json` (сейчас v2.0.0).

Голосовые команды

Wake word: «Коннор»
  • Коннор, запусти макрос 1 — выполнение макроса #1
  • Коннор, сделай скриншот — полный экран + OCR + копирование текста
  • Коннор, сделай скриншот области — выделение области мышью (прозрачный оверлей) + OCR + копирование
  • Коннор, спроси у DeepSeek ... — запрос в DeepSeek и показ ответа в “DeepSeek Chat”
  • Коннор, напомни через 5 минут про встречу — одноразовое напоминание
  • Коннор, напомни через 10 минут ежедневно про работу — ежедневное
  • Коннор, напомни через 1 час еженедельно про спорт — еженедельное

Горячие клавиши (по умолчанию)

  • Ctrl+Shift+T — показать/скрыть окно
  • Ctrl+Shift+M — заглушить/включить звук
  • Ctrl+Shift+B — открыть браузер
  • Ctrl+Shift+R — запустить распознавание голоса
  • Ctrl+Shift+Q — выход
Назначение можно изменить в Settings → “Горячие клавиши”.

Скриншоты и OCR

PNG сохраняются в:
./screenshots/screenshot_YYYY-MM-DD_HH-mm-ss.png
Распознанный текст копируется в буфер обмена.

DeepSeek Chat

  • Ключ API задаётся в Settings → “DeepSeek API”
  • История диалогов сохраняется в store
  • Ответ можно озвучивать через SpeechSynthesis
